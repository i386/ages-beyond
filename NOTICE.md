# Civilization IV: Ages Beyond

Ages Beyond is a Civilization IV: Beyond the Sword mod project based on MMod,
which itself builds on K-Mod and includes additional assets and components from
other Civ IV modding projects.

The repository is private while the project foundation is being prepared. Before
public release, verify attribution, redistribution permissions, and license
requirements for the inherited MMod, K-Mod, BUG, BAT, Varietas Delectat, map,
music, movie, and other bundled assets.
